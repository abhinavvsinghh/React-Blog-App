# Blog Post App

This React application allows users to manage blog posts. Users can view, add, edit, delete, and like blog posts.

## Features

- Show the list of blog posts.
- View details of a blog post.
- Add a new blog post.
- Edit an existing blog post.
- Delete a blog post.
- Like a blog post.

## Technologies Used

- React
- Redux
- Context API
